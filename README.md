# Dynamic-Excel-Dashboard-To-Do-List
This project is a dynamic Excel dashboard application developed to analyze, filter, and visualize task data using Microsoft Excel.  The dashboard updates automatically when the data changes and provides a clear overview of task status, priorities, and timelines.
